## Prerequisites
- Node.js

## Code setup
Install dependencies
```bash
npm ci
```
Build bundles
```bash
npm run build
```

## Live example
Run application server
```bash
node ./server/app-server.js
```
Run static server (in new terminal tab)
```bash
node ./server/static-server.js
```
Open in a browser
```bash
http://localhost:3000
```
## Development
Run application server
```bash
node ./server/app-server.js
```
Run development server (in new terminal tab)
```bash
npm run dev
```
